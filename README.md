# 轻博客 - 微信小程序前端

一个基于微信小程序的轻量级博客平台，用户可以浏览文章、搜索内容、通过标签筛选文章，并支持发布文章、评论互动、收藏点赞等社交功能。

## 项目结构

```
├── app.js                     # 小程序入口（全局数据、未读消息管理）
├── app.json                   # 应用配置（页面注册、底部Tab栏）
├── project.config.json        # 微信开发者工具配置
│
├── pages/
│   ├── index/                 # 首页 - 文章列表、分类筛选、搜索、消息入口
│   ├── search/                # 搜索 - 关键词搜索、热门/最新文章、文章归档
│   │   └── archives/          # 归档结果
│   ├── feature/               # 专题 - 按标签组合筛选文章
│   ├── login/                 # 登录
│   │   ├── login              # 登录页（微信一键登录 / 账号密码）
│   │   ├── account_login      # 账号密码登录
│   │   ├── register/          # 注册（两步）
│   │   └── forgetpwd/         # 找回密码（两步）
│   ├── postdetail/            # 文章详情（点赞、收藏、上下篇、分享）
│   ├── addpost/               # 创建/编辑文章（富文本、图片上传）
│   ├── applyauthor/           # 申请创作者权限

│       ├── editprofile/       # 编辑资料
│       ├── completeprofile/   # 完善信息
│       ├── hasaccount/        # 已有账号
│       ├── profilesecurity/   # 账号安全

│
└── assets/
```

## 功能特性

### 内容浏览
- **首页**: 分类文章列表，支持分类切换、关键词搜索、上拉加载更多、下拉刷新
- **搜索页**: 关键词搜索（输入防抖），展示最新/最热文章，按年月归档浏览
- **专题页**: 按标签组合筛选文章，实时显示匹配数量

### 文章互动
- **文章详情**: 富文本渲染、作者头像预览、标签点击搜索、点赞/收藏（需登录）、上下篇导航、微信分享
- **文章发布**: 作者可创建/编辑文章，包含标题、描述、分类、标签、正文及图片上传（最多6张），支持存草稿

### 用户系统
- **登录方式**: 微信一键登录、账号密码登录
- **注册/找回密码**: 分步骤流程
- **个人中心**: 资料查看编辑、账号安全、修改密码/邮箱
- **消息通知**: Tab栏未读红点提醒
- **创作者申请**: 提交申请理由等待审核

### 工程特性
- **HTTP 请求封装**: 统一鉴权、Token 自动刷新与请求队列重试
- **登录守卫**: 对需要登录的操作进行拦截引导
- **权限控制**: 区分游客/已登录/创作者角色
- **审核开关**: 通过后端 `appstatus` 接口控制"发布"按钮的显示

## 技术栈

- **框架**: 微信小程序原生框架
- **HTTP**: 自定义 `request` 封装（支持 `access/refresh` Token 双令牌机制）
- **渲染**: `mp-html` 第三方组件渲染富文本内容
- **后端 API**: 基于 Django/Django REST Framework（部署于 `yodragon.cn`）

## 开发

1. 使用微信开发者工具打开项目根目录
2. `project.config.json` 中已配置好基础设置（ES6 转译、代码压缩等）
3. 后端接口地址在 `utils/request.js` 中通过 `BASE_URL` 配置

```
// utils/request.js
const BASE_URL = 'https://yodragon.cn';   // 生产环境
// const BASE_URL = 'http://127.0.0.1:8000'; // 本地开发
```
